- **触发机制**：监听 `main` 分支推送及 `v*` 标签，支持手动触发。
- **多矩阵构建**：利用 `strategy.matrix` 并行处理 macOS (Intel/ARM) 和 Windows (x64/ARM) 四种目标架构。
- **资源预置**：在 Rust 编译前通过脚本动态下载并注入 Node.js 运行时及 Sidecar 二进制文件至 `src-tauri/resources`。
- **版本策略**：区分正式发行版（Tag）与 Nightly 预发布版；Nightly 采用“独立归档 Tag + 滚动 Nightly Tag”的双轨制，确保 Updater 始终指向最新稳定构建。
- **安全集成**：集成 Apple API Key 进行代码签名与公证，使用 Tauri Signing Key 生成更新清单。
- **同步作业**：`sync-nightly` 任务在构建完成后自动维护滚动标签，并从归档版本复制产物。
