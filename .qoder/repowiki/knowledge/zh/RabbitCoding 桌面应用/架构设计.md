- **前端层**：React + Vite 构建，通过 `@tauri-apps/api` 与后端通信，采用 Ant Design 和 Tailwind CSS 实现 UI。
- **后端层**：Rust (Tauri v2) 作为核心运行时，通过 `invoke_handler` 暴露系统级能力（如文件系统操作、Git 信息获取、SQLite 持久化）。
- **Sidecar 架构**：集成 Node.js Sidecar (`sidecar/`) 处理复杂的 AI Agent 逻辑，主进程通过 IPC 与其通信。
- **模块化设计**：Rust 端按功能拆分为 `wiki`（知识库）、`voice`（语音识别）、`auth`（认证）、`db`（数据持久化）等模块，前端通过自定义 Hooks（如 `useAgent`, `useWorkspaces`）封装业务逻辑。
