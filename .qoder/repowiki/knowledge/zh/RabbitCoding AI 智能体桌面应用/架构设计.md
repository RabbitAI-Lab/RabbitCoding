- **分层架构**：前端 (React/TS) 通过 Tauri Invoke 调用 Rust 后端；Rust 管理 Sidecar (Node.js) 进程并通过 JSON-lines 协议通信。
- **Sidecar 桥接**：Node.js Sidecar 封装 Claude Agent SDK，处理流式事件与 MCP 工具调用，将结果经 Rust 转发至前端。
- **Wiki 生成管线**：Rust 后端直接实现 AI 驱动的 Wiki 生成（Anthropic API + tool_use 循环），通过全局任务队列管理并发与断点续传。
- **插件生态**：通过 `CLAUDE_CONFIG_DIR` 环境变量隔离插件配置，支持 GitNexus、ECC、Claude-Mem 等插件的动态安装与管理。
- **状态同步**：利用 Tauri Events 实现后端向前端的实时状态推送（如 Wiki 进度、Agent 流式输出）。
