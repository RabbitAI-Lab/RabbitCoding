- **入口与布局**：`src/main.tsx` 挂载 `App.tsx`，采用左侧 `Sidebar`（工作区/任务列表）与右侧 `ContentArea`（聊天/文件/终端）的经典 IDE 布局。
- **状态管理层**：通过自定义 Hooks 实现逻辑分层。`useWorkspaces` 管理持久化数据（SQLite + localStorage 降级），`useAgentContext` 提升 Agent 通信状态至应用顶层以防止组件卸载中断流式响应。
- **通信层**：`useAgent` 封装 Tauri IPC (`invoke`/`listen`)，处理与后端 Sidecar 进程的 JSON 命令交互及事件监听，内置看门狗机制处理静默超时。
- **UI 组件层**：按功能域划分为 `agent`（消息流渲染）、`files`（文件树与编辑器）、`settings`（模型与插件配置）及 `terminal`（终端视图）。
- **国际化与主题**：`i18n` 模块提供上下文支持，`useTheme` 结合 Ant Design 的 `ConfigProvider` 实现深色模式同步。
