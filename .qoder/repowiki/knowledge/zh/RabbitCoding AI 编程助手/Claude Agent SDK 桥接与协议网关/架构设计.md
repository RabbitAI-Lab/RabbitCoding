- **入口层**：`src/index.ts` 基于 `node:readline` 监听 stdin，解析 JSON-lines 命令并分发给 handler。
- **核心逻辑层**：`src/agent.ts` 封装 `@anthropic-ai/claude-agent-sdk`，管理 `activeQueries`（AbortController）和 `pendingToolRequests`（AskUserQuestion 交互），将 SDK 的 AsyncGenerator 转换为标准化的流式事件。
- **协议定义层**：`src/protocol.ts` 使用 Zod 或 TS 接口严格定义双向通信契约（InboundMessage / AgentEvent）。
- **构建与资源层**：`scripts/setup-resources.mjs` 负责 esbuild 打包及平台原生二进制（claude.exe/claude）的提取与部署，确保生产环境下的隔离执行。
