- **混合进程架构**：采用 Tauri v2 的多进程模型，由 Rust 主进程（`src-tauri`）负责系统级交互与生命周期管理，React 渲染进程（`src`）负责 UI 呈现，Node.js Sidecar 子进程（`sidecar`）作为专用计算单元桥接 Claude Agent SDK。
- **跨语言通信契约**：前端通过 `@tauri-apps/api` 调用 Rust 命令；Rust 后端通过 `std::process::Command` 启动并监控 Sidecar 进程，双方通过标准输入输出（stdin/stdout）交换 JSON-lines 格式的结构化数据，实现流式响应与会话状态同步。
- **统一资源编排**：根目录通过 `pnpm-workspace.yaml` 管理前端与 Sidecar 的 TypeScript 依赖，并通过 `package.json` 中的 `setup:resources` 脚本协调 Sidecar 的资源预构建，确保多语言环境下的开发一致性。
