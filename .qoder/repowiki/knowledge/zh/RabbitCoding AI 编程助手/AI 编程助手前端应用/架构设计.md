- **技术栈**: 采用 React + TypeScript 构建 UI，Vite 作为构建工具，Ant Design (antd) 及 @ant-design/x 为组件库，通过 @tauri-apps/api 与 Rust 后端进行 IPC 通信。
- **状态管理**: 使用自定义 Hooks（如 `useWorkspaces`, `useAgentContext`）结合 Context API 管理全局状态。工作区数据优先持久化至 SQLite，支持 localStorage 降级备份，并实现双层防抖保存策略。
- **核心模块**:
  - `components/agent`: 负责 Agent 对话流展示，支持流式文本渲染、思维链显示、工具调用卡片及消息回滚编辑。
  - `components/sidebar`: 实现工作区与 Rabbit（任务会话）的层级导航及管理。
  - `components/settings`: 模块化设置中心，涵盖模型配置、MCP 服务、技能管理及网络诊断等面板。
  - `hooks`: 封装业务逻辑，包括 Agent 生命周期管理、语音输入、国际化及主题切换。
- **交互设计**: 支持侧边栏与设置面板的可拖拽调整宽度，集成暗色模式同步机制，并通过 `PetWindow` 支持独立宠物窗口视图。
