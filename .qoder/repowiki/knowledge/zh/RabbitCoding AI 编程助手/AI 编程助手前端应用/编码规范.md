- 自定义 Hooks 返回状态对象或 Context Provider，用于封装复杂业务逻辑（如 `useWorkspaces` 管理 CRUD 及持久化，`useAgentContext` 管理 Sidecar 通信）。
- 组件采用 Tailwind CSS 实用类进行样式定义，并广泛使用 `dark:` 前缀实现暗色模式适配。
- 异步数据加载或后端调用使用 `invoke` 函数，并配合 `try-catch` 进行错误处理及降级逻辑（如 DB 失败回退 localStorage）。
- 列表项或动态内容使用自定义 ID 生成器 (`generateId`) 确保唯一性，并在状态更新时采用不可变数据模式（Immutable updates）。
