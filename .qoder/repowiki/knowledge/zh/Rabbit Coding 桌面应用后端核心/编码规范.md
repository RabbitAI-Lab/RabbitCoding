- 所有 Tauri 命令函数均使用 `#[command]` 宏标记，并统一返回 `Result<T, String>` 以向前端传递错误信息。
- 耗时操作（如文件索引、网络探测、CLI 安装）均在 `tokio::task::spawn_blocking` 或独立 `std::thread` 中执行，避免阻塞主 UI 线程。
- 跨平台兼容性通过条件编译属性（如 `#[cfg(target_os = "windows")]`）处理，涵盖路径分隔符、命令行工具选择（nslookup vs dig）及通知机制。
- 前后端数据交互结构体统一使用 `#[serde(rename_all = "camelCase")]` 进行序列化，确保 Rust 蛇形命名与前端 JS 驼峰命名的无缝对接。
