- **应用入口与路由**：`src/main.tsx` 根据 URL 参数区分主窗口与宠物悬浮窗；`src/App.tsx` 作为根组件，通过 `view` 状态管理主界面、设置页、插件市场、待办事项及知识库页面的切换。
- **全局状态与上下文**：采用 Context API 封装核心业务逻辑。`AgentProvider`（`useAgentContext.tsx`）在顶层管理 Sidecar 进程生命周期及流式消息分发；`CodebaseIndexProvider` 管理代码索引状态；`AuthProvider` 处理认证；`I18nProvider` 提供多语言支持。
- **核心交互层**：`ContentArea.tsx` 是主工作区，集成 `@ant-design/x` 的 `Sender` 组件实现聊天输入，通过 `useAgent` Hook 与 Rust 后端通信，并动态渲染 `AgentChat` 流式对话流或欢迎页。
- **功能模块化**：`components/` 目录下按功能域划分，包括 `agent/`（对话渲染）、`wiki/`（知识库管理与 AI 生成管线前端编排）、`settings/`（模型、API Key、MCP 配置）以及 `pet/`（桌面宠物交互）。
- **持久化与工具**：自定义 `useLocalStorage` Hook 统一管理用户偏好（如主题、模型配置、代理设置）；`utils/` 提供 Spec 生成、通知及平台适配等纯函数逻辑。
