- **入口层**：`src/index.ts` 负责初始化 readline 接口，监听 stdin 解析 `InboundMessage`，并将处理结果以 JSON-lines 格式写入 stdout。
- **核心逻辑层**：`src/agent.ts` 封装 `@anthropic-ai/claude-agent-sdk`，将 SDK 的 AsyncGenerator 转换为事件流，管理活跃查询（AbortController）及工具请求（如 AskUserQuestion）的异步挂起与恢复。
- **协议定义层**：`src/protocol.ts` 使用 TypeScript 接口严格定义双向通信的消息结构，包括查询命令、流式增量（delta）、工具调用及系统状态。
- **依赖方向**：入口层依赖核心逻辑层执行具体业务，两者共同依赖协议层确保数据契约一致；通过 `zod` 进行运行时类型校验增强稳定性。
