- **前端层**：React + TypeScript 应用（`src/`），通过 Vite 构建，使用 Ant Design 与 TailwindCSS。
- **后端层**：Rust 核心（`src-tauri/`），利用 Tauri v2 提供系统级能力（文件系统、通知、窗口管理）及 SQLite 持久化。
- **Sidecar 架构**：内置 Node.js 运行时与 `claude-sidecar` 子进程（`sidecar/`），负责执行复杂的 AI Agent 逻辑并与主进程通信。
- **多窗口设计**：包含主工作区窗口（`main`）与透明宠物挂件窗口（`pet`），后者支持动态鼠标穿透交互。
