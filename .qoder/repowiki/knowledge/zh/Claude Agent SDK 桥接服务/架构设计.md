- **通信层**：基于 `stdin/stdout` 的 JSON-lines 协议（`protocol.ts`），由 `index.ts` 负责命令分发与生命周期管理。
- **核心逻辑**：`agent.ts` 封装 `@anthropic-ai/claude-agent-sdk`，将 SDK 的 AsyncGenerator 转换为前端可消费的流式事件（如 `text_delta`, `tool_use`）。
- **扩展机制**：支持通过 `CLAUDE_CONFIG_DIR` 动态发现并注入本地插件（如 claude-mem），并提供自定义 MCP 服务器（如 WriteSpec）以增强 Agent 能力。
- **交互控制**：内置 `AskUserQuestion` 拦截器，通过 `respond_tool_request` 实现异步人机交互；支持 Spec 生成场景下的工具权限管控（如拦截 ExitPlanMode）。
